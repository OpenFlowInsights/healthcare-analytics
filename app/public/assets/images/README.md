## Image Guidelines

**Style:**
- Clean, modern, professional
- Use OFI color palette (navy, blue, teal accents)
- Avoid stock photos of people pointing at screens
- Prefer abstract visualizations, real dashboards, or geometric designs

**Technical:**
- Format: JPG for photos, PNG for screenshots with text
- Compression: Optimize for web (<500KB per image)
- Responsive: Provide 2x versions for retina displays if needed

**Alt Text (Must Include):**
- All images must have descriptive alt text for accessibility
- Example: "MSSP shared savings dashboard showing spending trends and attribution data"
